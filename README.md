# tamara-portfolio
🚀 Professional Personal Portfolio
personal resume website built as part of the CCSW321 - Web Development course at the University of Jeddah. This project showcases my skills, education, and projects as a Software Engineering student.

🛠️ Built With
HTML5: Semantic structure for better accessibility and SEO.

CSS3: Custom styling using CSS Grid and Flexbox (No external frameworks like Bootstrap were used to ensure design uniqueness).

Responsive Design: Fully optimized for Mobile, Tablet, and Desktop views using Media Queries.
